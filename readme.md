Once the **BIOS Boot Sequence** is complete, use the command line to control the OS.

```bash
root@2026:~# help          # [SYSTEM] Lists all available commands
root@2026:~# celebrate     # [EVENT]  Initiates 2026 Fireworks + Audio Protocol
root@2026:~# game          # [APP]    Opens the Dual Game Launcher
root@2026:~# socials       # [NET]    Connects to Developer Profiles

root@2026:~# quote-life    # [DATA]   Fetches Wisdom from the Database
root@2026:~# music         # [AUDIO]  Generates a melody via Synth
```
